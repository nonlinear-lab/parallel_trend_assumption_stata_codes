# parallel_trend_assumption_stata_codes
This repository contains Stata codes for the parallel trend assumption
0 Stata codes for data preparation to construct variables from the census data of the American Community Survey (ACS) 
1 Stata codes for Variance Inflation Factor (VIF) and parallel trend analysis
2 Stata codes for event study, predictive margin and coefficient plot
3 Stata codes for placebo test for false year, placebo test for false group
4 Stata codes for placebo simulation analysis
